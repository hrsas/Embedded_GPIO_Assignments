# Problem 10: Display Diagonal LEDs from Right-Top to Left-Bottom

**Objective:**  
Display a diagonal line of LEDs starting from the **top-right corner** to the **bottom-left corner** on the 8x8 Dot Matrix.

**Files Included:**  
- `src/sketch.ino` → Code for right-top to left-bottom diagonal  
- `wokwi/diagram.json` → MAX7219 schematic diagram  
- `wokwi/wokwi-project.txt` → Wokwi metadata  

**Simulation Link :**  
[Open in Wokwi](https://wokwi.com/projects/445862770994478081)

**Steps to Run:**  
1. Open in Wokwi and upload code.  
2. Run simulation.  
3. View diagonal LED pattern.

**Expected Output:**  
- LEDs at positions `(0,7), (1,6), (2,5)...(7,0)` are ON.  
- They form a diagonal from **top-right → bottom-left**.
