# Problem 7: Press 7th Switch to Sequentially Glow 5th, 3rd, 0th, and 1st LEDs

**Objective:**  
When **7th switch** (PB6) is pressed, LEDs **5, 3, 0, and 1** on Port A glow sequentially with a delay.

**Files Included:**  
- `src/sketch.ino`  
- `wokwi/diagram.json`  
- `wokwi/wokwi-project.txt`

**Simulation Link:**  
[Open in Wokwi](https://wokwi.com/projects/444042348413516801)

**Steps to Run:**  
1. Open simulation.  
2. Press 7th switch.  
3. LEDs at positions 5 → 3 → 0 → 1 glow sequentially.

**Expected Output:**  
LEDs glow in order 5, 3, 0, 1 with visible delay and reset to OFF afterward.
